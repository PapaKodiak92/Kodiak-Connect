# Kodiak Connect v1.13.6

This debug patch improves RTC call support and call stability across desktop and mobile.

## What’s fixed

- Fixed WebRTC startup failures on browsers or app containers that do not expose the standard `RTCPeerConnection` global.
- Added fallback handling for WebRTC peer connection support.
- Improved voice-call startup checks so unsupported devices fail cleanly instead of half-starting a call.
- Improved mobile-to-mobile, mobile-to-desktop, and desktop-to-mobile call compatibility.
- Improved camera behavior during voice calls so one person can enable camera without forcing the other person to enable theirs.
- Improved call renegotiation when camera is turned on or off during an active call.

## Notes

- Users should update before testing calls again.
- This is a debug-focused RTC/call reliability patch.
- Windows installer signing/trust improvements are still handled separately.
