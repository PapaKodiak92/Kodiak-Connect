# Kodiak Connect v1.13.6

This debug patch improves voice and video call behavior across desktop, mobile, and Linux builds.

## What’s fixed

- Improved RTC compatibility for mobile-to-mobile, mobile-to-desktop, desktop-to-mobile, and Linux-to-desktop calls.
- Added missing Linux media dependencies for microphone/WebRTC support in packaged `.deb` builds.
- Fixed call startup handling for browsers or app containers that do not expose standard WebRTC globals.
- Improved speaking indicators during active calls.
- Improved local camera preview when a user turns camera on during a voice call.
- Improved camera toggling so one person can enable video without forcing the other person to enable theirs.
- Improved call renegotiation when camera is turned on or off mid-call.

## Notes

- Linux users should install the new `.deb` build before retesting calls.
- Users should update before testing calls again.
- This is a debug-focused call reliability patch.
- Windows installer signing/trust improvements are still handled separately.
